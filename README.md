# Delicias Cubanas — Auténtica Comida Criolla Cubana

Página web oficial del restaurante **Delicias Cubanas** en Holguín, Cuba.

## Características

- Hero con efectos visuales (patrón de losetas cubanas, hojas de palma flotantes, parallax 3D)
- Carrusel interactivo con imágenes reales del menú (efecto Ken Burns + grano cinematográfico)
- Sección "Nuestra Esencia" destacando tradición criolla, servicio y reconocimiento
- Calificaciones por estrellas con datos reales de reseñas
- Menú de platos principales con diseño de cards (CSS exacto proporcionado)
- Testimonios reales de Google Reviews
- Sección de contacto con Google Maps embed y teléfonos
- Totalmente responsivo
- Animaciones de entrada con IntersectionObserver

## Stack

- HTML5 + CSS3 + JavaScript vanilla
- Google Fonts (Playfair Display + Inter)
- Google Maps Embed
- SVG inline para iconos y efectos
